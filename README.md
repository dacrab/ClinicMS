# Σύστημα Διαχείρισης Κλινικής

CN5004 · Προχωρημένος Προγραμματισμός · 2025–26

JavaFX desktop εφαρμογή για διαχείριση ιατρών, ασθενών και ραντεβού.

## Λειτουργίες

- **Ιατροί** – CRUD, αναζήτηση
- **Ασθενείς** – CRUD, ιατρικό ιστορικό, αναζήτηση
- **Ραντεβού** – Προγραμματισμός (αποτροπή διπλοκράτησης), ολοκλήρωση, ακύρωση, φιλτράρισμα
- **Αποθήκευση** – CSV αρχεία (`data/`)

## Τεχνολογίες

- Java 21, JavaFX 21, Maven
- OOP: Doctor, Specialist, Patient, Appointment
- Singleton DataStore, CSV File I/O

## Εκτέλεση

```bash
mvn clean compile
mvn javafx:run
```

Απαιτήσεις: JDK 21+, Maven 3.8+
